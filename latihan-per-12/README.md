# Dashboard Informasi Publik

Mini project latihan REST API pertemuan 12.

## Fitur

- Mengambil data publik dari Open-Meteo, BMKG, ExchangeRate API, dan Random Facts.
- Menampilkan data dalam card, table, dan preview JSON.
- Auto refresh 30 detik, 1 menit, atau 2 menit.
- Error handling per API agar dashboard tetap berjalan jika salah satu layanan gagal.
- Validasi real untuk nama kota, latitude, dan longitude.

## Cara Menjalankan

Buka `index.html` di browser. Koneksi internet dibutuhkan karena UI library dan API memakai CDN/layanan publik.
